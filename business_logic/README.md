- 公文流转业务逻辑


## 教材业务逻辑

- 添加教材 (同一教材添加过一次后，不能重复添加)

::: tip
1.教材名称

2.出版社名称

3.作者

4.版别

5.选择课程 (教材和课程绑定)

6.教材类型  专业教材  通用教材  选读教材

7.教材介绍 
     
:::


- 教材数据的计算 (实时计算)
::: tip

一年级的教材数量是根据下一年的预招计划的实招人数 + n%的浮动

二年级的教材数量是根据一年级在校人数 + n%的浮动

其他年级的教材以此类推 

教材是和课程绑定，查找该课有那些专业在读，查找该专业的当前学期的总人数就是在校人数

:::
