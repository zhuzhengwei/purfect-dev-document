# banner接口

## 获取banner

- 接口地址: api/banner/getBanner

- 请求数据

| 参数名   | 是否必须  | 参数类型 | 说明      |
| -------- | :-------: | -----:   | -----: |
| posit    | Yes       | int      | 位置    |

- 响应数据
```json
{
    "code": 1000,
    "message": "OK",
    "data": [
        {
            "id": 1,
            "type": 2,
            "title": "标题",
            "image_url": "图片URL",
            "content": "文字内容",
            "external": "https://www.baidu.com",
        }
    ]
}   
```

- Banner(资源位的位置说明)
    - 0: 首页
    - 1: '招生资源位'
    - 2: '招生资源位'
    - 3: '培训资源位'
    - 4: '招生资源位'
    - 5: '购物资源位'
    - 6: '美食资源位'
    - 7: '超市资源位'
    - 8: '其他店铺'
    - 9: '天天特价资源位'
    - 10: '品牌闪购资源位'
    - 11: 就业资源位
    - 12: 教师首页

- Banner(类型说明)
    - 0: 无跳转及其他任何操作
    - 1: 图文
    - 2: URL: 跳转到某个网址
